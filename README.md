# Online Calculator

A simple, modern, and fully functional online calculator built with HTML, CSS, and JavaScript.

## Features

- ✨ Clean and modern UI with gradient design
- 🧮 Basic arithmetic operations: Addition, Subtraction, Multiplication, Division
- ⌨️ Full keyboard support
- 🎯 Decimal numbers support
- 🗑️ Delete and Clear buttons
- ❌ Error handling for division by zero
- 📱 Responsive design

## Live Demo

Click [here](https://your-github-username.github.io/calculator) to use the calculator online!

## How to Use

### Online
1. Visit the live demo link above
2. Click buttons or use your keyboard to perform calculations

### Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/calculator.git
   ```
2. Open `index.html` in your web browser
3. Start calculating!

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Enter numbers |
| `+` `-` `*` `/` | Operators |
| `.` | Decimal point |
| `Enter` or `=` | Calculate result |
| `Backspace` | Delete last digit |
| `Escape` | Clear all (AC) |

## Technologies Used

- HTML5
- CSS3 (with Flexbox & Grid)
- Vanilla JavaScript (ES6 Classes)

## Features Breakdown

### Operations
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)

### Buttons
- **AC**: Clear all inputs
- **DEL**: Delete last character
- **=**: Calculate result
- **Number & Decimal**: Input digits and decimals

## Installation & Deployment

### GitHub Pages Deployment
The project uses GitHub Actions for automatic deployment:
1. Push changes to the `main` branch
2. GitHub Actions automatically deploys to GitHub Pages
3. Access your calculator at: `https://YOUR_USERNAME.github.io/REPO_NAME`

### Manual Deployment
1. Go to repository Settings
2. Navigate to Pages
3. Set Source to `gh-pages` branch
4. Save
